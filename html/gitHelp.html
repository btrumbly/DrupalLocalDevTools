Basic Git Reference
  • Clone
    git clone https://github.com/[account]/[repo].git
    git clone https://github.com/usft/usfleettracking.com.git
    Git clone will clone the repo to your local disk.
  • Show Branch
    git branch
    Git branch show all the branches for the repo.
  • New Branch
    git branch [new branch name]
    git branch bugFix
    This will create a new branch on your local repo.
  • Switch Branch
    git checkout [branch name]
    git checkout bugFix
    This will change your branch.
  • Status
    git status
    This will show the current branch you are on as well as any changes to your branch or repo. Red will show items that have been changed. Green will show items added to your commit.
  • Add
    git add .
    This will add all un staged changes to your commit. You can also just add a single file by passing the file name in place of the period.
  • Commit
    git commit -m "[description of your commit]"
    git commit -m "bug fix for seo page'
    This will create a new commit to your local disk.
  • Push
    git push
    This will push your commit up to github. Note: if it is a locally created branch you will be prompted to enter an alternative command. Copy and paste this and hit return.
  • Pull
    git pull
    This will pull all changes down from github to your local copy. It is a good idea to do this on your master branch before starting work or creating a new branch.
  • More git reference can be found here https://git-scm.com/docs
